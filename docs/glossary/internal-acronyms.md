# Internal Acronyms Glossary

This glossary supports ShellDoc Assistant retrieval for internal delivery, QA, release, product, and engineering terminology.

## EoSB

Possible meaning:
- End of Sprint Build

Context:
Used in sprint-end release, QA validation, build handoff, and release candidate documentation.

Confidence:
Medium unless a source document defines it directly.
